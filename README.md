## 简介

洛谷保存站开源代码。

## 特性

- 基于 Node.js 22 实现
- 前端采用 Nunjucks + SemanticUI
- 后端使用 Express + MySQL

## 快速开始

```bash
git clone https://github.com/laikit-dev/luogu-saver.git
cd luogu-saver
npm install
node app.js
```
