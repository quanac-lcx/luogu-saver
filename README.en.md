## Introduction

Open source code for Luogu Saver's frontend.

## Features

- Built with Node.js 22
- Frontend: Nunjucks + SemanticUI
- Backend: Express + MySQL

## Getting Started

```bash
git clone https://github.com/laikit-dev/luogu-saver.git
cd luogu-saver
npm install
node app.js
```
